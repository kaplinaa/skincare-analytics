# D2C Skincare Analytics 

Аналитический проект: исследование продаж D2C-бренда косметики.

##  Основные метрики

- Клиентов: 454
- Заказов: 1,250  
- Выручка: 1.12 млн ₽
- Средний чек: 940 ₽
- Возвраты: 6.79%

##  Технологии

Python • Pandas • SQL • Power BI

## Файлы проекта

- 'skincare_analytics.ipynb' — анализ данных в Jupyter Notebook
- 'queries.sql' — SQL-запросы
- 'skincare_dashboard.pbix' — Power BI дашборд
- 'Аналитическая записка.docx' — отчет с выводами и рекомендациями
- 'dashboard_guide.md' — руководство по дашборду
